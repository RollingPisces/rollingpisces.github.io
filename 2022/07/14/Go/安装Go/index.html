    1. 下载安装包

    官网：https://golang.org/dl/

    根据操作系统选择相应的安装包进行安装。

    1.1. Linux系统特别指导

    1. 获取安装包地址
      对于linux系统，右键单击安装包的链接，选择Copy link address（复制链接地址）：

    2. 在Linux机器上wget刚刚复制的address

       wget https://go.dev/dl/go1.18.3.linux-amd64.tar.gz

    3. 解压缩
      执行tar将上面安装包解压到/usr/local目录下：

      sudo tar -C /usr/local/ -zxvf go1.18.3.linux-amd64.tar.gz

    4. 打开/etc/profile（或者$HOME/.profile）设置GOROOT和PATH

      sudo vi /etc/profile
# 最后一行添加：
export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin
# 保存退出后source一下：
source /etc/profile

    2. 检查Go环境是否安装成功

    go version

    执行上述命令查看go的版本号，如果成功显示版本号则安装成功。
